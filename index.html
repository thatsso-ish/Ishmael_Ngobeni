2023 Nov - 2023 Dec

Crickek Predictive Model

Used technologies

  • Jupyter

About

This project focuses on building a cricket prediction model to forecast the likelihood of a player getting out based on various features. The process begins with data preprocessing, where cricket batting data from a CSV file is cleaned and prepared for analysis. Key steps include handling missing values, converting data types, and scaling features using a custom scaler class. The core of the project involves training a logistic regression model using the preprocessed data. The model's performance is evaluated, and its coefficients are analyzed to understand feature impacts. Additionally, the trained model and scaler are saved for future use in making predictions. For enhanced interaction and visualization, Streamlit is used to create a web application that displays results and insights. The project also integrates with a MySQL database to store prediction results, ensuring a seamless workflow from data preprocessing to model deployment and visualization.
